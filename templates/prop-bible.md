### Locked visual description

A self-contained paragraph that can paste straight into any Seedance Static Description or Banana Pro scene plate. Cover:

- **Form & scale:** dimensions, proportions, silhouette
- **Materials:** every surface — matte, glossy, brushed, weathered, oxidized, painted, decaled
- **Colors:** primary, secondary, accents (hex values if precise grading matters)
- **Mechanical detail:** seams, hinges, rivets, hydraulics, hardware
- **State:** pristine / lived-in / battle-worn / damaged — and how
- **Signature features:** glowing parts, moving parts, anything the camera lingers on
- **For creatures:** anatomy, posture, locomotion, sounds it makes
